Para entrar al proyecto primero:

1. Poner composer install
2. copy .env.example .env 
3. npm install 
4. npm run build   
5. npm run dev
6. php artisan key:generate
7. php artisan migrate 
8. php artisan serve
      
Esos son los pasos que se hace para poder entrar al proyecto.
